# MyNewsHub v2.0 — Your Personal News Dashboard

## Deploy to Vercel (Free, ~10 minutes)

1. Unzip this folder
2. Go to vercel.com → create free account
3. Click "Add New Project" → drag and drop this folder
4. Click Deploy — done!
5. Bookmark your new URL on phone + desktop
